

   ===============
    ProteinShaker
   ===============

   To start ProteinShaker simply double click the jar file.

   From the command line use 'java -jar ProteinShaker-X.Y.Z.jar'. 
   (Replace X.Y.Z with the ProteinShaker version number.)

   For help with runnng ProteinShaker please see:
   http://protein-shaker.googlecode.com
